# warehousePLAY
Proyecto creado para el aprendizaje del Framework Play con JAVA.

### ¿Cómo arrancar el proyecto?
Arrancar el proyecto con:
```bash
sbt run
```
y abrir en el explorador [http://localhost:9000/](http://localhost:9000/)
